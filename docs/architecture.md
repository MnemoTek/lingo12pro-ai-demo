# Lingo12PRO Architecture

Lingo12PRO is a multilingual AI voice assistant built using Python.

The system integrates speech recognition, prompt engineering, and the Gemini AI model to provide multilingual interaction and translation.

## Components

• Speech Recognition – captures and converts voice input to text  
• Prompt Engineering – controls AI response behavior  
• Gemini API – generates intelligent responses  
• Text-to-Speech – converts responses into audio  
• GUI Interface – allows user interaction
