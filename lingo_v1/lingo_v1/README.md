# Lingo v1 – Initial Prototype

This is the first prototype of the Lingo AI assistant.

The initial version was a simple Python script designed to run in the terminal using PyCharm.

Characteristics of version 1:

• Single language interaction  
• Command line interface  
• Speech recognition input  
• Text-to-speech responses  
• Basic integration with the Gemini API  

This prototype served as the starting point for the development of the full system.

Later versions introduced:

• graphical interface  
• multilingual support  
• advanced prompt engineering  
• improved architecture

The final system evolved into **Lingo12PRO**, a multilingual AI voice assistant.
