## Lingo12PRO – Multilingual AI Voice Assistant

Lingo12PRO is a Python-based AI assistant capable of multilingual interaction using speech recognition, AI processing, and text-to-speech synthesis.

The system integrates the Gemini API for advanced language understanding and dynamic prompt engineering.

---

## Features

- Multilingual AI assistant  
- Voice input via microphone  
- Real-time speech recognition  
- AI-powered translation  
- Text-to-speech output  
- Multiple language modes  
- Prompt engineering for AI control  

---

## Tech Stack

- Python  
- Tkinter GUI  
- Gemini API  
- SpeechRecognition  
- gTTS  
- Google Speech Recognition  

---

## System Architecture

User Input (Voice / Text)  
↓  
Speech Recognition  
↓  
Prompt Engineering  
↓  
Gemini AI Processing  
↓  
Response Generation  
↓  
Text-to-Speech  
↓  
User Interface
